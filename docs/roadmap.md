# Forge — Roadmap

## Phase 1 — MVP (~3 months)

**Goal:** Prove the full loop: agent order → distributed execution → verification → settlement

- `forge_run` MCP tool (OSS)
- Central Broker (centralized mock)
- Worker Daemon (runs on surplus PCs)
- pytest replication + majority vote verification
- Off-chain ledger for rewards
- **Demo: Claude Code autonomously dispatches pytest to Forge**

## Phase 2 — v1 (~6 months)

- Expand Worker participants
- Reputation / Stake / Slash mechanics
- Support for general CPU batch tasks

## Phase 3 — v2 (~12 months)

- On-chain settlement (Base or Solana)
- Wasm standard execution environment
- General agent runtime support (OpenAI Agents SDK, AutoGen, etc.)

## Long-term Vision

Become the standard protocol for inter-agent compute economy via MCP/A2A.

> Forge is where the AI economy runs.
