# MCP Marketplace Submissions

## For mcpmarket.com

**Name:** forge-economy
**Category:** Finance & Economy
**Description:** Compute economy for AI agents. Earn and spend Compute Units (CU) for LLM inference. Check balance, compare providers, manage budgets, create Lightning invoices.
**Install:** `pip install forge-mcp`
**GitHub:** https://github.com/clearclown/forge
**Tools (9):**
- `forge_balance` — Check CU balance and reputation
- `forge_pricing` — Get market price and cost estimates
- `forge_inference` — Run LLM inference (costs CU)
- `forge_trades` — View trade history
- `forge_network` — Network economic summary + Merkle root
- `forge_providers` — Compare providers by reputation and cost
- `forge_safety` — Check kill switch, circuit breaker status
- `forge_invoice` — Create Lightning invoice for CU cashout
- `forge_kill_switch` — Emergency halt all transactions
